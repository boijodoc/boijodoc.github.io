Filebeat 패치 노트

Filebeat v7.6 패치 노트

주요 변경 사항

모든 비트

  • 사용자 정의 정책에 대한 업그레이드 경험을 향상 시키려면 기본 ILM 정책에서 버전 정보를 제거하십시오. 14745
  • setupcmd를 실행 setup.ilm.overwrite하면 사용자 지정 정책에 대한 향상된 지원을위한 설정이 적용됩니다. 14741
  • 새 라이센스 엔드 포인트 및 새 형식을 사용하도록 x-pack 라이센스 프로그램 코드를 정리하십시오. URL / _xpack / license를 / _license로 바꿉니다. 15091
  • 문서 ID 필드의 이름이 @ metadata.id에서 @ metadata._id로 변경되었습니다. 15859
  • 동일한 데이터 경로를 가진 두 개의 Beat 인스턴스를 동시에 실행할 수 없습니다. 14069

Filebeat

  • CEF 확장은 이제 CEF 안내서에 정의 된 데이터 유형에 매핑됩니다. 14342

Journalbeat

  • 깨진 대시 보드를 제거하십시오. 15288

Metricbeat

  • 지표와 차원 모두에 대한 Cloudwatch 지표 세트 매핑을 업데이트합니다. 15245

Packetbeat

  • TLS : ECS에 맞게 필드가 변경되었습니다. 15497
  • TLS : send_certificates 및 include_raw_certificates 옵션의 동작이 변경되었습니다. 15497
